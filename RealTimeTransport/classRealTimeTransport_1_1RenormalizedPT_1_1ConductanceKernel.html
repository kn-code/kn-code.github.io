RealTimeTransport::RenormalizedPT::ConductanceKernel Class Reference
2. Renormalized perturbation theory

#include <RealTimeTransport/RenormalizedPT/ConductanceKernel.h>

This class computes the conductance kernel via the renormalized perturbation theory. This kernel can be used to compute the conductance \( dI_r/d\mu_r \) of reservoir \( r \) without numerical differentiation, which is numerically more stable. The memory and current kernel must be computed first in order to use this class. A cross conductance \( dI_r/d\mu_{r'} \) can be obtained by first computing the current kernel for resrevoir \( r \), and then the conductance kernel for reservoir \( r' \).
